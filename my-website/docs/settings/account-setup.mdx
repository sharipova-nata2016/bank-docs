# Как настроить аккаунт в приложении NovaBank

1. Перейдите в раздел **Профиль**.  
2. Нажмите **Редактировать**.  
3. Введите адрес электронной почты.  
Нажмите **Далее**.  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
<img alt= "Шаг 1" height = "600" src = "../img/account_setup_1.png" width = "300"/>  

2. Выберите тип учетной записи.  
Нажмите **Далее**.  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
<img alt= "Шаг 2" height = "600" src = "../img/account_setup_2.png" width = "300"/>    

3. Добавьте адрес регистрации: адрес, город или регион, индекс.  
Нажмите **Далее**.  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
<img alt= "Шаг 3" height = "600" src = "../img/account_setup_3.png" width = "300"/>    

4. Введите полное имя.  
Нажмите на поле **Дата рождения**, выберите дату в календаре.  
Нажмите **Далее**.  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
<img alt= "Шаг 4" height = "600" src = "../img/account_setup_4.png" width = "300"/>
